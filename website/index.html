The control plane for agent actions.

Self-hosted and lightweight. Every tool your agents call passes through one place to be approved, denied, and logged.

Install
Linux
$curl -L https://github.com/validmind/atryum/releases/download/0.0.2/atryum-linux -o atryum && chmod +x atryum
macOS
$curl -L https://github.com/validmind/atryum/releases/download/0.0.2/atryum-darwin-arm64 -o atryum && chmod +x atryum
